# My Neovim configuration

## Install neovim
```zsh
brew install neovim
```

## Install supporting packages
```zsh
brew install ripgrep fd
```

# Download my neovim configuration
```zsh
mkdir -p ~/.config/nvim
git clone git@github.com:avinashjoshi/neovim-config.git ~/.config/nvim
```
